# 💡 Gemini CLI 강화 기능 사용 설명서

본 워크스페이스는 `workspace-master` 및 전문가 툴킷이 장착된 강화된 상태입니다.

### ⌨️ 단축 명령어 (Quick Commands)
1. **`Status`**: 현재 모든 프로젝트의 기술 스택, 진행 상황, 활성 스킬을 한눈에 확인합니다.
2. **`Wrap up`**: 오늘의 모든 대화 요약 + 보안 스캔 + Git 푸시 + 윈도우 알림을 한 번에 실행합니다.
3. **`Help`**: 지금 이 가이드를 다시 불러옵니다.

### 🛡️ 지능형 자동 보호 (Active Guard)
- **실시간 보안 스캔**: 모든 파일 수정 및 Git 푸시 전, API 키 노출 여부를 제가 스스로 감시합니다.
- **명령어 보호**: `rm -rf` 같은 위험한 명령어를 치시면 실행 전 제가 먼저 경고를 드립니다.

### 🎨 전문가 설계 (Dev Expert)
- **UI 리팩토링**: "더 세련되게 바꿔줘"라고 하면 최신 Tailwind 패턴을 적용합니다.
- **Firebase 최적화**: 데이터 구조를 업계 표준 트랜잭션 방식으로 설계합니다.

### 🧠 맥락 관리 (Token Saver)
- 대화가 길어지면 제가 스스로 맥락을 압축하여 응답 속도를 최상으로 유지합니다.
